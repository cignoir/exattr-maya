Extra Attribute Editor v1.0.1

Mayaのカスタムアトリビュートを便利に管理するためのプラグイン

ダウンロード GitHub

主な機能

  • シーン全体のカスタムアトリビュートを高速スキャン
  • カスタムアトリビュートが設定されたノードのバッチ編集
  • フィルター・ソート機能搭載
  • カスタムアトリビュートが設定されたマテリアルからアサインされたポリゴンをまとめて選択

技術仕様

対応バージョン
Maya 2025 以降
