Extra Attribute Editor v1.0.1

Mayaのカスタムアトリビュートを便利に管理するためのプラグイン

ダウンロード GitHub

使い方

  • ダウンロードした exattr-maya.mll をプラグインフォルダ %USERPROFILE%\Documents\maya\2025\plug-ins\など にコピー
  • Maya の Plug-in Manager でプラグインをロード
  • メニューから Windows > General Editors > Extra Attribute Editor を選択
  • または MEL コマンド exAttrEditor -ui; で起動

主な機能

  • シーン全体のカスタムアトリビュートを高速スキャン
  • カスタムアトリビュートが設定されたノードのバッチ編集
  • フィルター・ソート機能搭載
  • カスタムアトリビュートが設定されたマテリアルからアサインされたポリゴンをまとめて選択
  • 割り当てポリゴンの抽出 (Extract Assigned Polygons):
    • マテリアルリストで右クリック > Extract Assigned Polygons... を選択
    • オプション機能:
      • New Object Name: 抽出後のオブジェクト名を指定
      • Keep Original Polygons: 元の面を残すか削除するかを選択
      • Assign Single Lambert Material: 抽出後にデフォルトマテリアルを割り当て
      • Combine Meshes: 複数の抽出結果を1つのメッシュに結合（UV維持）
    • 抽出されたオブジェクトは自動的に [名前]_grp にグループ化されます
